Fasd Keyboard v0.1

DCPU-16 Hardware Info:
    Name: Keyboard
    ID: 0xec402d15, version: 0x0001
    Manufacturer: 0xFA5D0000 (Fasd)

Description:
    Fasd Keyboard can deal with the input of stdin while running. It has a buffer which can store 0x10000 chars. And when a key was tapped, it will call an interrupt. Before use it, remember to enable it :)

Interrupt behavior:
    When a HWI is received by the Keyboard, it reads the A register and does one of the following actions:

0x0000: 
    Shutdown DCPU-16.
